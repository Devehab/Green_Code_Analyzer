Green Code Analyzer - Demo

How to Use the Extension

1

Install the Extension

After loading the extension into Chrome, you'll see the Green Code Analyzer icon in your toolbar.

[Screenshot of extension icon in Chrome toolbar]

2

Click on the Extension Icon

Click on the Green Code Analyzer icon to open the popup interface.

[Screenshot of popup interface]

3

Paste Your Code

Paste your code into the text area and select the programming language from the dropdown menu.

[Screenshot of code input and language selection]

4

Analyze the Code

Click the "Analyze Code" button to start the analysis. The extension will send the code to the Google Gemini API for processing.

[Screenshot of loading state]

5

View the Results

Review the analysis results, including the efficiency score, carbon impact, and recommendations for improvement.

[Screenshot of analysis results]

Using the Context Menu

1

Select Code on a Webpage

Find code on any webpage (like GitHub, StackOverflow, etc.) and select it.

[Screenshot of selected code on a webpage]

2

Right-Click and Select "Analyze Code for Energy Efficiency"

Right-click on the selected code and choose the "Analyze Code for Energy Efficiency" option from the context menu.

[Screenshot of context menu]

3

View the Results

The extension will open automatically and analyze the selected code.

[Screenshot of analysis results]

Configuring the Extension

1

Access the Settings

Click the settings icon in the popup or right-click the extension icon and select "Options".

[Screenshot of settings icon]

2

Configure Your Settings

Enter your Google Gemini API key, set the default programming language, and enable/disable auto-analyze.

[Screenshot of settings page]

3

Save Your Settings

Click the "Save Settings" button to apply your changes.

[Screenshot of save button]
